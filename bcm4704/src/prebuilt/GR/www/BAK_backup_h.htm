Hilfe: Einstellungen sichern

Auf dieser Seite können Sie die aktuellen Einstellungen des Routers sichern, wiederherstellen und löschen.

Nachdem Sie den Router eingerichtet haben, sollten Sie Ihre Einstellungen sichern, um im Falle einer Störung darauf zurückgreifen zu können. Beim Sichern werden die Einstellungen als Datei auf Ihrem Computer gespeichert. Sie können die Routereinstellungen aus dieser Datei wiederherstellen.

So erstellen Sie eine Sicherungsdatei der aktuellen Einstellungen:

  1. Klicken Sie auf Sichern.
  2. Wenn Ihr Webbrowser nicht so eingestellt ist, dass heruntergeladene Dateien automatisch gespeichert werden, wählen Sie das Verzeichnis aus, in dem Sie die Datei speichern möchten, passen Sie (falls gewünscht) den vorgegebenen Dateinamen an und klicken Sie auf Speichern.
    Wenn Ihr Webbrowser heruntergeladene Dateien automatisch speichert, wird die Datei unter dem Namen NETGEAR_WNR834Bv2.cfg im Downloadverzeichnis des Browsers auf der Festplatte gespeichert.

WICHTIG! Stellen Sie keine Internetverbindung her, schalten Sie den Router nicht aus, fahren Sie den Computer nicht herunter und führen Sie keine anderen Vorgänge aus, während die Routereinstellungen wiederhergestellt oder gelöscht werden. Warten Sie damit UNBEDINGT so lange, bis der Router neu gestartet wurde! Dies sollte nur etwa eine Minute dauern. Wenn die Test-LED aufgehört hat zu blinken, warten Sie noch einige Sekunden, bevor Sie fortfahren.

So stellen Sie Einstellungen aus einer Sicherungsdatei wieder her:

  1. Klicken Sie auf Durchsuchen.
  2. Wählen Sie die zuvor erstellte Sicherungsdatei aus (standardmäßig lautet der Dateiname NETGEAR_WNR834Bv2.cfg).
  3. Klicken Sie auf Wiederherstellen.
    Nachdem die Einstellungen wiederhergestellt wurden, wird eine entsprechende Meldung angezeigt. Der Router wird neu gestartet. Das dauert etwa eine Minute.
    WICHTIG! Stellen Sie keine Internetverbindung her, schalten Sie den Router nicht aus, fahren Sie den Computer nicht herunter und führen Sie keine anderen Vorgänge aus, bis der Router neu gestartet wurde. Wenn die Test-LED aufgehört hat zu blinken, warten Sie noch einige Sekunden, bevor Sie fortfahren.
  4. Schließen Sie das Meldungsfenster.

So löschen Sie die aktuellen Einstellungen und setzen den Router auf die Werkseinstellungen zurück:

  • Klicken Sie auf Löschen.
    WICHTIG! Stellen Sie keine Internetverbindung her, schalten Sie den Router nicht aus, fahren Sie den Computer nicht herunter und führen Sie keine anderen Vorgänge aus, bis der Router neu gestartet wurde. Wenn die Test-LED aufgehört hat zu blinken, warten Sie noch einige Sekunden, bevor Sie fortfahren.

Nachdem Sie die aktuellen Einstellungen des Routers gelöscht haben, lautet das Routerpasswort password, die LAN-IP-Adresse ist 192.168.1.1 und der Router fungiert als DHCP-Server im LAN und als DHCP-Client am WAN-Port.
