Hilfe: Router aktualisieren

Auf der Seite Router aktualisieren können Sie neue Versionen der Routersoftware installieren.

Besuchen Sie die Netgear-Website, um eine neue Version der Routersoftware von der NETGEAR-Website herunterzuladen. Anschließend müssen Sie die heruntergeladene Datei zunächst entpacken (entzippen oder extrahieren), um sie auf den Router übertragen zu können.

WICHTIG! Nachdem Sie auf Hochladen geklickt haben, dürfen Sie die Übertragung der Software auf den Router und das Neustarten des Geräts KEINESFALLS unterbrechen. Wenn Sie eine Unterbrechung dieser Vorgänge nicht mit absoluter Sicherheit ausschließen können, klicken Sie auf Abbrechen, um die bisherige Routersoftware beizubehalten.

So aktualisieren Sie die Routersoftware:

  1. Gehen Sie zu www.NETGEAR.com oder www.NETGEAR.de und laden Sie dort die aktuellste Software herunter.
  2. Entpacken (dekomprimieren) Sie die heruntergeladene Datei, sofern dies nicht automatisch geschieht.
    Bevor Sie fortfahren, sollten Sie die Release Notes (Versionshinweise) lesen.
  3. Klicken Sie auf Durchsuchen.
  4. Suchen Sie die soeben heruntergeladene und entpackte Datei, und wählen Sie diese aus.
  5. Klicken Sie auf Hochladen, um die Software an den Router zu übertragen.
    Nachdem die Software in den Router geladen wurde, wird das Gerät neu gestartet.
    Hinweis: Stellen Sie keine Internetverbindung her, schalten Sie den Router nicht aus, fahren Sie den Computer nicht herunter und führen Sie keine anderen Vorgänge aus, bis der Router neu gestartet wurde! Warten Sie noch einige Sekunden ab, nachdem die Bereitschaftsleuchte zu blinken aufgehört hat, bevor Sie mit dem nächsten Schritt fortfahren.
  6. Klicken Sie auf Routerstatus und überprüfen Sie anhand der angezeigten Firmwareversion, ob die neue Software erfolgreich installiert wurde.

WICHTIG! Im Anschluss an die Aktualisierung der Firmware muss der Router unter Umständen neu konfiguriert werden. Weitere Informationen hierzu finden Sie in den Release Notes (Versionshinweisen).

Sollte die Aktualisierung der Routersoftware in der hier beschriebenen Weise nicht durchgeführt werden können, finden Sie im Referenzhandbuch auf der Router GearBox-CD Hinweise zu anderen Verfahren.
