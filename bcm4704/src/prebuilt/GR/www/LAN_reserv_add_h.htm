Hilfe: Adressreservierung

Wenn Sie für einen Computer im LAN eine reservierte IP-Adresse festlegen, wird diesem bei jedem Zugriff auf den DHCP-Server dieselbe IP-Adresse zugewiesen. Indem Sie einem Server eine reservierte IP-Adresse zuweisen, ist dieser immer unter derselben Adresse erreichbar.

IP-Adressen reservieren

In der Tabelle IP-Adressen reservieren werden die angeschlossenen Geräte im Netzwerk angezeigt, denen reservierte IP-Adressen zugewiesen werden können.

So reservieren Sie eine IP-Adresse:

  1. Aktivieren Sie in der Tabelle IP-Adressen reservieren die Optionsschaltfläche des Computers, dem Sie eine reservierte IP-Adresse zuweisen möchten.
  2. Wenn der gewünschte Computer nicht in der Tabelle angezeigt wird, klicken Sie auf Aktualisieren.
  3. Wenn der Computer noch immer nicht in der Tabelle IP-Adressen reservieren erscheint, geben Sie dessen IP-Adresse, MAC-Adresse und Gerätenamen manuell ein.
  4. Klicken Sie anschließend auf die Schaltfläche Hinzufügen.

So übernehmen oder verwerfen Sie die Änderungen:

Klicken Sie auf Hinzufügen, um dem ausgewählten Computer eine reservierte IP-Adresse zuzuweisen.
Klicken Sie auf Abbrechen, um die Seite ohne Speichern der Änderungen zu verlassen.
