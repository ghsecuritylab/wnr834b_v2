Hilfe: Liste der zugriffsberechtigten WLAN-Karten

Standardmäßig kann jeder WLAN-fähige Computer mit der richtigen SSID auf das WLAN zugreifen. Zur Erhöhung der Sicherheit können Sie den WLAN-Zugang auf einzelne Computer mit vorgegebenen MAC-Adressen beschränken.

Auf der Seite WLAN-Karten einrichten finden Sie eine Liste der WLAN-fähigen Computer, deren MAC-Adressen für den Zugriff auf den Router freigegeben wurden. Ob diese Computer auf das WLAN zugreifen können, hängt außerdem von den Einstellungen für SSID und WEP auf der Seite WLAN-Konfiguration ab.

Zugriffssteuerung aktivieren

    1. Klicken Sie auf das Kontrollkästchen Zugriffssteuerung aktivieren, um den WLAN-Zugang auf Computer mit vorgegebenen MAC-Adressen zu beschränken
    2. Klicken Sie auf die Schaltfläche Übernehmen, um die Änderungen zu speichern und zur Seite WLAN-Konfiguration zurückzukehren.

Hinweis: Wenn Sie die Option Zugriffssteuerung aktivieren wählen, die Liste für die Zugriffssteuerung jedoch keine Einträge enthält, kann von keinem Computer aus auf das WLAN zugegriffen werden.

So richten Sie die Liste für die Zugriffssteuerung ein:

    1. Klicken Sie auf die Schaltfläche Hinzufügen, um zum Menü Zugriff einrichten zu gelangen. Hier finden Sie eine Liste der bereits freigegebenen WLAN-Karten mit den jeweiligen MAC-Adressen.
    2. Wenn der gewünschte Computer in der Liste angezeigt wird, können Sie dessen MAC-Adresse in das Eingabefeld übernehmen, indem Sie auf die entsprechende Optionsschaltfläche klicken. Sie können die MAC-Adresse des zugriffsberechtigten Computers auch manuell eingeben. Bei den meisten WLAN-Geräten ist die MAC-Adresse auf einem Aufkleber auf der Unterseite angegeben.
    3. Wenn kein Gerätename angezeigt wird, können Sie einen beschreibenden Namen für den hinzuzufügenden Computer eingeben.
    4. Nachdem Sie die MAC-Adresse eingegeben haben, klicken Sie auf Hinzufügen, um zur Liste für die Zugriffssteuerung zurückzukehren.
    Hinweis: Wiederholen Sie die Schritte 1 bis 4 für jeden zugriffsberechtigten Computer.
    5. Klicken Sie auf Zugriffssteuerung aktivieren.
    6. Klicken Sie auf die Schaltfläche Übernehmen, um die Änderungen zu speichern und zur Seite WLAN-Konfiguration zurückzukehren.

So bearbeiten Sie einen Eintrag in der Liste für die Zugriffssteuerung:

    1. Klicken Sie auf die Optionsschaltfläche des Computers, den Sie bearbeiten möchten.
    2. Klicken Sie auf die Schaltfläche Bearbeiten.
    3. Ändern Sie den Gerätenamen oder die MAC-Adresse.
    4. Klicken Sie anschließend auf die Schaltfläche Übernehmen.
    5. Klicken Sie auf die Schaltfläche Übernehmen, um die Änderungen zu speichern und zur Seite WLAN-Konfiguration zurückzukehren.

So löschen Sie einen Eintrag aus der Liste für die Zugriffssteuerung:

    1. Klicken Sie auf die Optionsschaltfläche des Computers, den Sie aus der Liste löschen möchten.
    2. Klicken Sie auf die Schaltfläche Löschen.
    3. Klicken Sie auf die Schaltfläche Übernehmen, um die Änderungen zu speichern und zur Seite WLAN-Konfiguration zurückzukehren.

So speichern oder verwerfen Sie die Änderungen:

Klicken Sie auf Übernehmen, wenn Sie mit dem Anpassen der Einstellungen fertig sind.
Klicken Sie auf Abbrechen, um zu den vorigen Einstellungen zurückzukehren.
