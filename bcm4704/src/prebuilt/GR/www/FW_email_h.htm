Hilfe: E-Mail

Wenn ein Benutzer in Ihrem LAN (Local Area Network) versucht, auf eine gesperrte Site zuzugreifen, kann automatisch eine entsprechende Benachrichtigung versendet werden. Protokolle sind Listen aller aufgerufenen Internetadressen. Wenn Sie Benachrichtigungen und Protokolle per E-Mail erhalten möchten, füllen Sie dieses Formular aus.

Sie können die Protokolle jederzeit manuell über die Seite Protokolle abrufen. Wenn Sie Benachrichtigungen und Protokolle nicht per E-Mail erhalten möchten, lassen Sie die Felder auf dieser Seite leer.

So empfangen Sie Benachrichtigungen und Protokolle per E-Mail:

  1. Aktivieren Sie das Kontrollkästchen E-Mail-Benachrichtigung aktivieren.
  2. Geben Sie in das Feld Server für den Mailversand den SMTP-Mailserver Ihres Internetdienstanbieters (z. B. smtp.internetdienstanbieter.net) ein.
    Wenn Sie dieses Feld frei lassen, werden keine Benachrichtigungen und Protokolle gesendet.
    Tipp: Die Angaben in diesem Bereich entsprechen denen, die Sie beim Einrichten Ihres E-Mail-Programms eingegeben haben. Wenn Sie sich nicht mehr an diese Angaben erinnern können, übernehmen Sie diese aus den Einstellungen Ihres E-Mail-Programms.
  3. Geben Sie in das Feld E-Mail-Adresse des Empfängers die Adresse ein, an die die Benachrichtigungen und Protokolle gesendet werden sollen. Sie können Ihre eigene E-Mail-Adresse oder die eines anderen Benutzers verwenden.
    Geben Sie die vollständige E-Mail-Adresse ein (z. B. Susi.Schulze@internetdienstanbieter.net).
  4. Wenn Sie bei jedem Zugriffsversuch auf eine gesperrte Internetadresse sofort eine Benachrichtigung per E-Mail erhalten möchten, aktivieren Sie das Kontrollkästchen Benachrichtigungen sofort senden.
  5. Wenn keine Protokolle per E-Mail versendet werden sollen, wählen Sie in der Liste unter Protokolle nach diesem Zeitplan senden die Option Nie.
    oder
    Wählen Sie eine der anderen Optionen in der Liste, um die Protokolle regelmäßig per E-Mail zu versenden.
    • Wenn Sie die Option Wöchentlich wählen, müssen Sie den gewünschten Wochentag angeben.
    • Wenn Sie die Option Wöchentlich oder Täglich wählen, müssen die Uhrzeit für den Versand der E-Mails angeben.
  6. Klicken Sie auf Übernehmen, um die Änderungen an den Router zu übermitteln.

Zeitzone

Wählen Sie hier die örtliche Zeitzone aus. Diese Angabe wird für den termingerechten E-Mail-Versand entsprechend dem Zeitplan benötigt.

Bei Bedarf können Sie außerdem das Kontrollkästchen Automatisch an Sommer-/Winterzeit anpassen aktivieren.
