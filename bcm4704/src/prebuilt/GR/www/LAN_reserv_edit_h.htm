Hilfe: Reservierte Adressen bearbeiten

Wenn Sie für einen Computer im LAN eine reservierte IP-Adresse festlegen, wird diesem bei jedem Zugriff auf den DHCP-Server dieselbe IP-Adresse zugewiesen. Indem Sie einem Server eine reservierte IP-Adresse zuweisen, ist dieser immer unter derselben Adresse erreichbar.

So bearbeiten Sie eine reservierte IP-Adresse:

    1. Ändern Sie die IP-Adresse, die MAC-Adresse oder den Gerätenamen.
    2. Klicken Sie anschließend auf die Schaltfläche Übernehmen.

So übernehmen oder verwerfen Sie die Änderungen:

Klicken Sie auf Übernehmen, um Ihre Änderungen in die Tabelle Reservierte Adressen zu übernehmen.
Klicken Sie auf Abbrechen, um nicht gespeicherte Änderungen rückgängig zu machen.
