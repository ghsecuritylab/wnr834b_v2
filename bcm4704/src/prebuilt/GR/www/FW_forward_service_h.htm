Hilfe: Benutzerdefinierte Dienste bei der Portkonfiguration

So richten Sie eine Anwendung, ein Spiel oder einen Dienst ein:

  1. Geben Sie den Dienstnamen in das entsprechende Feld ein.
  2. Geben Sie die Nummer des ersten Ports im Portbereich in das Feld Erster Port ein.
  3. Wenn die Anwendung nur einen Port verwendet, geben Sie dieselbe Portnummer in das Feld Letzter Port ein.
  4. Wenn die Anwendung einen Bereich aus mehreren Ports verwendet, geben Sie die Nummer des letzten Ports in diesem Bereich in das Feld Letzter Port ein.
  5. Geben Sie im Feld IP-Adresse des Servers die IP-Adresse des Computers ein.
  6. Klicken Sie auf Übernehmen.

So bearbeiten Sie einen Dienst in der Liste:

  1. Passen Sie die Angaben in den Feldern Dienstname, Erster Port, Letzter Port und IP-Adresse des Servers an.
  2. Klicken Sie auf Übernehmen.
