Hilfe: Passwort festlegen

Auf der Seite Passwort festlegen können Sie das Passwort für den Zugriff auf die Einstellungen des Routers ändern. Dieses Passwort hat nichts mit dem Passwort zu tun, das Sie für den Internetzugang benötigen.

So ändern Sie das Passwort:

  1. Geben Sie das bisherige Passwort ein.
    Das standardmäßig vorgegebene Passwort für den Router lautet password.
  2. Geben Sie das neue Passwort ein und bestätigen Sie es, indem Sie es nochmals in das Feld Neues Passwort wiederholen eingeben.
    Hinweis: Notieren Sie sich das Passwort an sicherer Stelle.
  3. Klicken Sie auf Übernehmen, um das Passwort zu ändern, oder auf Abbrechen, um das bisherige Passwort beizubehalten.

Hinweis: Wenn Sie die Routereinstellungen gesichert haben, sollten Sie diesen Vorgang nun erneut durchführen, damit das geänderte Passwort in die Sicherungsdatei aufgenommen wird.
