Hilfe: Dienste sperren

Mit Dienste sperren können Sie den Internetzugriff durch bestimmte Benutzer in Ihrem lokalen Netzwerk anhand der jeweiligen IP-Adressen sperren. Sie können beispielsweise verhindern, dass ausgewählte oder alle Benutzer im Web surfen (also den http-Dienst sperren). Dienstsperren können Nie, Nach Zeitplan oder Immer aktiviert werden.

So aktivieren Sie die Dienstsperren:

  • Klicken Sie auf Nach Zeitplan, um die Dienstsperren entsprechend den Einstellungen auf der Seite Zeitplan zu aktivieren.
  • Klicken Sie auf Immer, um die Dienstsperren unabhängig von den Einstellungen auf der Seite Zeitplan zu aktivieren.

So deaktivieren Sie die Dienstsperren:

  • Klicken Sie auf Nie, um die Dienstsperren zu deaktivieren.

Diensttabelle

In dieser Tabelle sind alle gesperrten Dienste aufgelistet. Sie können nach Bedarf Einträge hinzufügen oder löschen.

So fügen Sie einen neuen Dienst hinzu:

  1. Klicken Sie auf die Schaltfläche Hinzufügen.
  2. Füllen Sie die Felder auf der daraufhin angezeigten Seite aus.
  3. Klicken Sie anschließend auf Übernehmen.

So bearbeiten Sie einen Dienst:

  1. Aktivieren Sie die Optionsschaltfläche des Dienstes, den Sie bearbeiten möchten.
  2. Klicken Sie auf die Schaltfläche Bearbeiten.
  3. Nehmen Sie die gewünschten Änderungen an den Einstellungen für den Dienst vor.
  4. Klicken Sie anschließend auf die Schaltfläche Übernehmen.

So löschen Sie einen Dienst:

  1. Klicken Sie auf die Optionsschaltfläche neben einer der Tabellenzeilen.
  2. Klicken Sie auf Löschen, um den gewählten Dienst aus der Tabelle zu entfernen.

So speichern oder verwerfen Sie die Änderungen:

Klicken Sie auf Übernehmen, um die Einstellungen zu übernehmen.
Klicken Sie auf Abbrechen, um die Seite ohne Speichern der Änderungen zu verlassen.
