Hilfe: WLAN-Karten einrichten

Verfügbare WLAN-Karten

In der Liste Verfügbare WLAN-Karten werden alle verfügbaren Computer mit WLAN-Unterstützung und deren MAC-Adressen angezeigt.

Wenn der gesuchte Computer in der Liste Verfügbare WLAN-Karten angezeigt wird, können Sie durch Klicken auf die entsprechende Optionsschaltfläche dessen MAC-Adresse übernehmen. Wenn der gesuchte Computer nicht angezeigt wird, können Sie auf die Schaltfläche Aktualisieren klicken, um die Liste der verfügbaren Computer mit WLAN-Unterstützung zu aktualisieren. Wenn der gesuchte Computer noch immer nicht angezeigt wird, können Sie die MAC-Adresse auch manuell eingeben. Gehen Sie dabei wie im Folgenden beschrieben vor.

Eintrag für WLAN-Karte

Wenn in der Liste Verfügbare WLAN-Karten keine Computer mit WLAN-Unterstützung angezeigt werden, können Sie den Gerätenamen und die MAC-Adresse des zugriffsberechtigten Computers manuell eingeben.

Hinweis: Die MAC-Adresse ist eine zwölfstellige Zeichenfolge, die Sie bei den meisten WLAN-Geräten einem Aufkleber an der Unterseite entnehmen können.

So übernehmen oder verwerfen Sie die Änderungen:

Klicken Sie auf Hinzufügen, wenn Sie alle gewünschten Änderungen an den Einstellungen vorgenommen haben.
Klicken Sie auf Abbrechen, um zu den ursprünglichen Einstellungen zurückzukehren.
Klicken Sie auf Aktualisieren, um die zum aktuellen Zeitpunkt angeschlossenen Geräte anzuzeigen.
