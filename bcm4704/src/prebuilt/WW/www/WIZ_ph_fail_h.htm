No Internet Connection Detected

The Setup Wizard wasn't able to access the Internet through the Cable/DSL modem.

Please verify the following:

  • The modem is connected to the external line.
  • The settings you entered are correct.
    Note: Make sure there are no typos. Many values are case-sensitive; therefore, PatAB is not the same as patab.

If you've corrected the problem, click Retry to have the router try again.

Click Cancel to return the beginning of the Setup Wizard.

If you continue to have problems:

  • Contact your ISP to verify the configuration information.
  • Read the Troubleshooting section in the Router Installation Guide.
  • On the Router Resouce CD, read the Troubleshooting Guide or the Troubleshooting section in the Reference Manual.
  • Contact NETGEAR Technical Support.
