Logs Help

A log is a detailed record of the Web sites that users on your network have accessed or attempted to access. If you have set up content filtering on the Block Sites page, the Logs page shows you when someone on your network tried to access a blocked site. If you have e-mail notification on, you'll receive these logs in an e-mail message. If you don't have e-mail notification set up, you can view the logs here.

  • Click Clear Log to delete all of the log entries.

  • Click Refresh to see the most recent access attempts.

  • Click Send Log to send the log file to your email account. This feature can be very useful for testing your email settings.
