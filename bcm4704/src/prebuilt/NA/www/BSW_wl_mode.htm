  Which wireless mode do you want to use?

  Selecting the right Wireless Mode optimizes the performance of your wireless network while reducing wireless interference. Select 130 Mbps if there are several strong wireless signals in your neighborhood. Select 270 Mbps if there are only few neighboring wireless networks and you want to get the maximum possible Wireless-N speeds.

Up to 130 Mbps: Neighbor Friendly Mode in presence of neighboring wireless networks.
Up to 270 Mbps: Performance Mode for Maximum possible Wireless-N speed.
